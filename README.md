# LangGraph Research Agent (Human-in-the-Loop)

基于 LangGraph 的研究代理，支持人工干预。可在 planner（研究计划）和 writer（报告草稿）阶段审核、修改后再继续执行。

## 功能特性

- **可控状态机**: 在关键节点自动中断，等待人工审核
- **灵活干预**: 支持 approve（批准）或 modify（修改指令）两种操作
- **状态持久化**: 使用 MemorySaver 保存执行状态
- **中文界面**: 系统提示和交互界面均为中文

## 安装

```bash
pip install -r requirements.txt
```

## 配置

在项目根目录创建 `.env` 文件：

```env
DASHSCOPE_API_KEY=your_dashscope_api_key
TAVILY_API_KEY=your_tavily_api_key
```

- `DASHSCOPE_API_KEY`: 阿里云 DashScope API 密钥（用于 Qwen3.5-plus 模型）
- `TAVILY_API_KEY`: Tavily 网页搜索 API 密钥

## 使用

### 运行研究代理

```bash
python -m src.main "AI"
```

### 交互流程

1. **Planner 阶段中断**: 显示生成的研究计划步骤
   - 输入 `approve` - 批准计划，继续执行
   - 输入 `modify: <指令>` - 修改计划（例如：`modify: 添加大语言模型相关内容`）

2. **Writer 阶段中断**: 显示生成的报告草稿
   - 输入 `approve` - 保存报告
   - 输入 `modify: <指令>` - 重新生成报告（例如：`modify: 请添加更多技术细节`）

3. **完成**: 报告保存至 `outputs/{topic}_report.md`

## 工作流程

```
┌─────────────┐
│    START    │
└──────┬──────┘
       ▼
┌─────────────┐
│   planner   │  ← 生成研究计划
└──────┬──────┘
       │ interrupt_before=["researcher"]
       ▼
┌─────────────┐
│  researcher │  ← 执行搜索
└──────┬──────┘
       ▼
┌─────────────┐
│    writer   │  ← 生成报告
└──────┬──────┘
       │ interrupt_before=["saver"]
       ▼
┌─────────────┐
│    saver    │  ← 保存报告
└──────┬──────┘
       ▼
┌─────────────┐
│     END     │
└─────────────┘
```

## 项目结构

```
src/
  main.py      # CLI 入口，交互式循环
  graph.py     # LangGraph StateGraph 定义
  nodes.py     # 节点实现：planner, researcher, writer, saver
  state.py     # ResearchState 状态定义
  config.py    # LLM 配置
  tools.py     # 工具函数
simple_react.py  # 遗留的 ReAct 代理
outputs/         # 生成的报告存放目录
```
